# Menu-de-Operacoes-Matematicas-em-Python
Sistema desenvolvido em Python que oferece uma interface interativa via terminal que permite ao usuário realizar diversas operações lógicas e aritméticas sobre dois valores de entrada. O sistema utiliza um laço de repetição contínuo para manter o menu ativo, permitindo múltiplas operações ou a redefinição dos dados de entrada sem interromper a execução.
